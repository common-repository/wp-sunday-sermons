This plugin adds sermon functionality to all WP Sunday child themes for the Genesis Framework.

== Description ==

The **WP Sunday - Sermons** plugin makes it easy to manage your church's sermons and sermon series. It adds a custom post type for sermons. As well as custom taxonomy for sermon series. Using the WP Sunday - Sermons widget, you can feature your latest sermons in any registered widget area of your site.

> <strong>Please Note:</strong><br>
> This plugin requires the [Genesis Framework by StudioPress](http://studiopress.com/), but it is neither endorsed nor supported by StudioPress.
> 
> To get the most out of this plugin it should be used with a [WP Sunday child theme](https://www.wpsunday.com/themes/).

== Installation ==

1. Upload the entire wp-sunday-sermons folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In your Widgets menu, simply drag the widget labeled "WP Sunday - Sermons" into a widgetized area
4. Configure the widget by choosing a title, sermon series, and the sermon content you would like to display

== Frequently Asked Questions ==

= Can I use both YouTube and Vimeo for my sermon videos? =

Yes, you can add a YouTube video to one sermon and a Vimeo video to another. If you try adding both to a single sermon the Vimeo video will override the YouTube video.

= Which video hosting services are supported? =

* Vimeo
* YouTube

= Will you be adding more video hosting options in the future? =

It is possible if enough users request a specific service, however we believe strongly in the value of simplicity.
